🧭 Tourist Safety Monitoring System

A full-stack web application built to help monitor and enhance the safety of tourists in real time, using live weather data to generate safety alerts and recommendations.

📌 Overview

This project was built as a mini-project by a 4-member team. It combines a web-based front end with a weather-data-driven backend to give tourists (and authorities monitoring them) timely, location-relevant safety information.

✨ Features


Real-time location and weather-based safety alerts using the OpenWeather API
Persistent user and location data storage with MongoDB
Clean, responsive front-end interface built with HTML/CSS
Safety recommendations generated based on live conditions


🛠️ Tech Stack

Layer Technology Front-end HTML, CSS Database MongoDB External API Open Weather API

🚀 How It Works


User location data is captured through the web interface.
The system queries the OpenWeather API for real-time weather conditions at that location.
Based on the weather data, the system generates relevant safety alerts/recommendations.
Data is stored in MongoDB for record-keeping and future reference.
